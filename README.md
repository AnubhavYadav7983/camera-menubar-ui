# 📸 Camera Menu Bar UI Component

An interactive, sleek, and responsive **Camera Menu Bar / Control Panel UI** designed for modern web applications, digital camera software, or vision system dashboards. This project focuses on building a clean user interface with intuitive control options, mode transitions, and smooth interactive states.



## 🛠️ Tech Stack
* **HTML5:** Semantic tags for structuring menus, control dials, and status indicators.
* **CSS3:** Flexbox layout, CSS variables for quick theme tweaking, custom icons styling, and custom UI sliders.
