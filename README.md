# 📸 Camera Menu Bar UI Component

An interactive, sleek, and responsive **Camera Menu Bar / Control Panel UI** designed for modern web applications, digital camera software, or vision system dashboards. This project focuses on building a clean user interface with intuitive control options, mode transitions, and smooth interactive states.

## ✨ Features
* **Interactive Control Bars:** Quick access toggles for camera parameters (e.g., Shutter Speed, ISO, White Balance, or Resolution).
* **Smooth Mode Switching:** Seamless transitions between Photo, Video, Panorama, and Pro Modes.
* **Modern Aesthetic:** Dark-themed frosted glass UI effect (Glassmorphism) tailored for camera and multimedia tools.
* **Responsive Layout:** Optimized to look and function perfectly on both smartphone viewports (mobile camera layout) and desktop monitoring screens.

## 🛠️ Tech Stack
* **HTML5:** Semantic tags for structuring menus, control dials, and status indicators.
* **CSS3:** Flexbox layout, CSS variables for quick theme tweaking, custom icons styling, and custom UI sliders.
* **JavaScript:** Handles interactive click events, active state changes across navigation menus, and real-time state logging.
